### What is Ambr?

---

Ambr's goal is to become _"the fastest blockchain platform based DAG and support smart contract over a secure, decentralized network."_

We've applied the philosophy of _"do one thing and do it well"_.

---

### How to build it

Linux
* install boost
* install cmake
* ./setenv.sh
* make
* run ambr
* http://127.0.0.1:8080
